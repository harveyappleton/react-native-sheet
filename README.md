# react-native-sheet

A cross platform, pure JS implementation of bottom sheets in React Native.

## Installation

```sh
npm install react-native-sheet
```

## Usage

```js
import { multiply } from "react-native-sheet";

// ...

const result = await multiply(3, 7);
```
